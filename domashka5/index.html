Селектори у css

  • По class (буде набиратись через крапку (.) у вікні Сss сторінки)
  • По id (По айді буде набиратись через решітку (#) у вікні Сss страниці)
  • По style ( на сторінці html пишуть атрібутом (style), буде важатись інлайновим чи через тег style) )

  • По назві елементу (Прикладом цього є елемент header цієї сторінки)

  • Через * для усіх елементів сторінки

Також по вложенності наприклад цього єлементу (div p), через пробіл обов'язково!
